# אתר MINOLA 🎁

אתר נחיתה סטטי לעסק MINOLA — מתנות, מיתוג אישי והדפסה על מוצרים.

## איך לראות את האתר במחשב

פשוט פותחים את `index.html` בדפדפן (לחיצה כפולה), או מריצים שרת מקומי:

```bash
cd site
python -m http.server 8000
# ואז נכנסים בדפדפן ל: http://localhost:8000
```

## קבצים

| קובץ | תפקיד |
|---|---|
| `index.html` | המבנה והתוכן של האתר |
| `style.css` | העיצוב (צבעים, גופנים, פריסה) |
| `script.js` | תפריט מובייל ושנה בפוטר |

## מה צריך להחליף לפרטים אמיתיים

- **מספר וואטסאפ** — מחפשים `972500000000` ומחליפים במספר האמיתי (פורמט בינלאומי, בלי 0 בהתחלה).
- **קישור אינסטגרם** — מחפשים `instagram.com/` ומוסיפים את שם המשתמש.
- **גלריה** — מחליפים את ריבועי "דוגמה 1/2/3..." בתמונות אמיתיות.
